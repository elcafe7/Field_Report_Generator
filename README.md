# Tactical Field Report Generator

A comprehensive, multi-format reporting tool designed for rapid tactical documentation and field situational awareness.

## 🚀 Overview

The **Tactical Field Report Generator** is a versatile web-based application that streamlines the creation of standardized reports. It provides a clean, user-friendly interface for generating critical reports in high-pressure environments.

### Supported Report Types
- **SALUTE:** Size, Activity, Location, Unit, Time, Equipment.
- **SALT:** Size, Activity, Location, Time (Simplified).
- **ACE:** Ammunition, Casualties, Equipment (Status report).
- **SITREP:** Situation Report for general status updates.

### Key Features
- **Centralized Dashboard:** Quick-access cards for all supported report formats.
- **Structured Data Entry:** Guided forms to ensure consistent data quality across all report types.
- **Instant Preview & Export:** 
  - **Copy to Clipboard:** Optimized for rapid relay via secure channels.
  - **Clean Formatting:** Standardized headers and timestamps for every report.
- **Privacy First:** 100% client-side operation. No data is ever transmitted to a server.
- **Modern UI:** Clean, Apple-inspired interface with intuitive navigation.

## 🛠️ Installation & Usage

### Direct Usage
1. Clone the repository or download `index.html`.
2. Open `index.html` in any modern web browser.

### Local Development / Hosting
```bash
git clone https://github.com/elcafe7/Field_Report_Generator.git
cd Field_Report_Generator
# Serve locally using Python
python3 -m http.server 9000
```

## 📖 How it Works
1. **Select Report:** Choose the required tactical report type from the main dashboard.
2. **Input Data:** Complete the guided fields specific to that report format.
3. **Review:** Verify the information in the real-time preview.
4. **Distribute:** Copy the formatted text for immediate transmission or print the report for a physical record.
